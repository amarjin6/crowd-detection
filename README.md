ğŸŒ **Crowd Detection is a real-time occupancy counting system.** Was created website and server part based on You only look once (YOLO) a state-of-the-art, real-time object detection system. Our model has several advantages over classifier-based systems. It looks at the whole image at test time so its predictions are informed by global context in the image.
